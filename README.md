# EsSalud 2026-2031 — Ayuda memoria para el más alto nivel

Propuesta de reforma estructural de EsSalud para el periodo de gobierno 2026-2031,
elaborada con el **Modelo GEMSES** (Carlos Pérez Pérez) sobre fuentes oficiales
(PEI EsSalud 2026-2030, Diagnóstico MTPE nov-2025, RENIPRESS-SUSALUD 2026,
Libro Blanco EsSalud/Banco Mundial, informe OIT de gobernanza, ComexPerú).

## Contenido

- **`index.html`** — Ayuda memoria pública (interactiva): mapas del Perú (IPRESS por nivel
  y población asegurada), gobierno y liderazgo, radiodiagnóstico descentralizado,
  financiamiento y las 22 acciones rápidas clasificadas por instrumento legal.
  → Publicada con GitHub Pages.
- **`privado/index.html`** — Dossier técnico completo (los dos documentos de texto,
  matriz legal detallada, tablas de datos y fuentes). **Cifrado con AES-256; requiere clave de acceso.**

El dossier viaja cifrado (AES-GCM + PBKDF2, descifrado en el navegador con WebCrypto),
por lo que su contenido no es legible en el repositorio sin la clave.

> Documento de trabajo con fines de asesoría.
